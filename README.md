# IT Support Ticket Dashboard

## Overview
A React-based dashboard for creating, tracking, updating, and deleting IT support tickets.

## Features
- Create support tickets
- Track ticket status: Open, In Progress, Resolved
- Track priority levels
- Delete tickets
- Dashboard summary cards
- Saves data using localStorage

## Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript
- localStorage

## Purpose
This project connects my IT support background with front-end development by showing how support requests can be organized and tracked in a simple dashboard.
